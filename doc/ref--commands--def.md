# `def`
<small>*Command*</small>  
**Synopsis**

```cpp
#[use]def <signature>
...
#enddef

```
```cpp
#[use]def <signature> = ...;

```


New macro definition, it will create an unnamed group on the inside.
The unnamed group is activated if `use` is in the command.
